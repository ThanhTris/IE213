# Kiểm Thử Tự Động (Tests)

Thư mục này chứa các bài kiểm thử tự động cho dự án IE213.

## Cấu Trúc

```
tests/
├── backend/     # Kiểm thử API backend
└── frontend/    # Kiểm thử giao diện frontend
```

## Chạy Kiểm Thử

### Backend Tests
```bash
cd backend
npm test
```

### Frontend Tests
```bash
cd frontend
npm test
```

### Tất cả Tests (từ thư mục gốc)
```bash
npm test
```
